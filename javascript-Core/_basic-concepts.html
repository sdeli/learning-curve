BASIC CONCEPTS

EXPRESSIONS AND STATEMENTS

An expression produces a value and can be written wherever a value is expected

  • statement performs an action
  • Loops and if statements are examples of statements
  • Wherever JavaScript expects a statement, you can also write an expression: expression statement

LITERALS

  • are fixed values that you literally provide in your application source, and are not variables.
  • 42
  • 3.14159
  • "To be or not to be"
