MOBILE FIRST WEBDESIGN

MOBILE FIRST - CONCEPT Websites are built in opposite direction: Mobile View => Tablet View => Desktop View -due to the higher traffic comes from mogile devices
Download website elements specified to viewport - prevent to download data user wont use
Load as quickly as possible on mobile

MOBILE FIRST - DESIGN CONTEXT - Designing around smaller screen
- Prioritize content
- Anticipate most common user actions
- Experts involver: Information Arcitect, Usability Expert, Graphic Designer

MOBILE FIRST - DEVELOPMENT CONTEXT - Website shouldnt involve data unnecessary on users viewport
- Treat the most essential viewport as baseline and develop upwards from there
- Quick load for every wievport
